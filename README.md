# ГолосРуки 2.0 — управление Android голосом

Голосовое управление смартфоном «как пальцами»: нажатия, свайпы, навигация,
ввод текста, запуск приложений, звонки, **SOS** и разблокировка экрана.
Распознавание русского — **офлайн** через Vosk. Для людей с ограниченными возможностями.

## Что нового в 2.0

- 🗣️ **Активация по имени «Иван».** После активации команды работают подряд.
  Через **30 секунд тишины** — сон, нужно снова сказать «Иван».
- 🔓 **Разблокировка экрана: «Иван привет».** Будит экран и снимает блокировку.
- 🎯 **Точность через грамматику Vosk** — распознаются только слова команд.
- 🆘 **SOS** — по слову «сос» (или «спасите»/«тревога») отправляется СМС с
  геолокацией и совершается звонок на заданный номер.
- 🔢 **Сетка 3×4 = 12 ячеек, двухуровневая.** «Сетка» → «нажми 7» (зум в ячейку) →
  «нажми 3» (точное попадание). Минимум слов — максимум точности.
- 👤 **Персональные команды** — «позвони жене», «открой телеграм» из
  редактируемого файла `app/src/main/assets/personal_commands.json`.
- ✍️ **Диктовка** — «диктовка» переключает в свободный ввод текста, «готово» — выход.

## Команды (после «Иван»)

| Сказать | Действие |
|---|---|
| иван | активировать |
| иван привет | разбудить и разблокировать экран |
| назад · домой · недавние · шторка | навигация |
| номера | пронумеровать кнопки |
| нажми N | нажать элемент/ячейку N |
| долгое нажатие N · двойное нажатие N | удержание · двойной тап |
| сетка | сетка 12 ячеек (двухуровневая) |
| листай вверх/вниз/влево/вправо | свайп |
| позвони <имя> | звонок контакту из конфига |
| открой <приложение> | запуск приложения |
| напиши <текст> | ввод в активное поле |
| диктовка … готово | свободный ввод текста |
| удали · ввод | стереть символ · Enter |
| сос · спасите · тревога | вызов помощи (СМС с гео + звонок) |
| стоп · слушай | пауза · продолжить |
| скрой · помощь | убрать наложение · подсказка |

## Перед сборкой: настройте personal_commands.json

Откройте `app/src/main/assets/personal_commands.json` и впишите **реальные**
номера и нужные приложения (имена контактов — в дательном падеже: «позвони КОМУ»):

```json
{
  "контакты": { "жене": "+79990000001", "врачу": "+79990000004" },
  "приложения": { "телеграм": "org.telegram.messenger" },
  "sos": { "номер": "+79990000000", "текст": "SOS! Нужна помощь." }
}
```

# Установка: GitHub + Termux (со смартфона)

Модель Vosk (~45 МБ) **скачивается автоматически** в GitHub Actions — в репозиторий
её класть не нужно.

### 1. Подготовка Termux (один раз)
```bash
pkg update && pkg upgrade -y
pkg install git openssh -y
# создать SSH-ключ и добавить его в GitHub (Settings → SSH keys):
ssh-keygen -t ed25519 -C "termux"
cat ~/.ssh/id_ed25519.pub      # скопировать и вставить в GitHub
```

### 2. Создать пустой репозиторий на GitHub
Например `golosruki`. Тип — private или public, без README.

### 3. Залить проект из Termux
Распакуйте архив проекта в Termux (например в `~/GolosRuki`), затем:
```bash
cd ~/GolosRuki
git init
git add .
git commit -m "GolosRuki 2.0"
git branch -M main
git remote add origin git@github.com:ВАШ_ЛОГИН/golosruki.git
git push -u origin main
```

### 4. Дождаться сборки
Откройте на GitHub вкладку **Actions** — workflow «Build APK» соберёт APK
(скачает модель, соберёт проект). Это 5–10 минут.

### 5. Скачать и установить APK
APK появится в разделе **Releases → latest → app-debug.apk**.
Скачайте на телефон и установите (разрешите «Установка из неизвестных источников»).

> При повторной установке поверх старой версии возможен конфликт подписи —
> сначала удалите предыдущую версию приложения.

### 6. Первый запуск
1. Кнопка 1 — выдать разрешения (микрофон, телефон, СМС, геолокация).
2. Кнопка 2 — включить службу «ГолосРуки» в спец. возможностях.
3. Кнопка 3 — запустить. В шторке появится «ГолосРуки активен».
4. Скажите **«Иван»** и проверьте: «Иван номера» → «нажми 1».

## Важно про разблокировку и надёжность

- 🔐 PIN / графический ключ / пароль **программно обойти нельзя** — это защита Android.
  Для полной разблокировки голосом установите блокировку экрана **«Нет»** или **«Свайп»**
  (Настройки → Безопасность). Тогда «Иван привет» будит и открывает экран.
- 🎙️ Vosk теряет точность в шуме — нужен хороший микрофон рядом.
- 🆘 Оставьте **физический запасной способ** позвать на помощь (большая кнопка / близкий рядом).
  Приложение — помощник, а не единственная «нитка жизни».
- ☎️ Штатные Google Ассистент / Алиса (звонки, СМС, умный дом) — хорошее дополнение.

## Развитие
- Связка с умным домом (Home Assistant / ESP32): свет, кондиционер, ТВ.
- Свой словарь под имена контактов для ещё большей точности.
- Wake-word на отдельной лёгкой модели для энергоэффективности.
