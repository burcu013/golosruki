# ГолосРуки — карта команд, состояний и логики (v6.52)

Как устроена логика: режимы, когда команда срабатывает, нужно ли «Иван», что делает, сколько держится, как выключается.

---

## 1. Состояния и режимы

| Режим | Что значит | Вход | Выход | Слышит без «Иван»? |
|---|---|---|---|---|
| 😴 **Сон** | Ждёт слово активации | Бездействие ~30 c | «Иван» | Только «Иван» (+ пара быстрых при выкл. экране) |
| 🎙 **Бодрствование** | Принимает команды подряд | «Иван» | Бездействие → Сон | Да |
| ✍️ **Диктовка** | Речь → текст в поле (дописывает к курсору) | «Иван напиши/диктовка» | «готово/конец/стоп/отмена/Иван» | Да |
| 🔢 **Диктовка цифрами** | Речь → цифры | «Иван цифры/номер телефона» | как диктовка | Да |
| 🎵 **Медиа-режим** | Плеер/листание без «Иван» | «Иван медиа» | «медиа», блокировка, выход | Да (медиа/листание) |
| 📞 **Звонок** | Идёт вызов (мобильный или в приложении) | вызов | завершение | **Нет — только «Иван…»** |
| 🧠 **Размышление ИИ** | Модель генерирует | после вопроса | приход ответа | Нет |
| 🗣️ **Озвучка ответа** | Иван проговаривает ответ | после генерации | конец речи или «Иван хватит» | **Только «Иван + хватит/стоп»** |
| ❓ **Захват ИИ-запроса** | Ждёт вопрос/текст | «Иван думай/спроси/сформулируй/поговорим» | фраза, «стоп/отмена/Иван», сон | Да |
| 💬 **Диалог с ИИ** | Беседа подряд | «Иван поговорим» | «хватит/достаточно/закончили/стоп» | Да |
| ⏸️ **Пауза** | Распознавание стоит | «молчи/замри» | «слушай/продолжи» | Только «слушай»/«Иван» |

## 2. Таймауты и удержания

| Что | Сколько | Сброс |
|---|---|---|
| Бодрствование → Сон | ~30 c бездействия | любая команда продлевает |
| Медиа-режим | бессрочно | «медиа», блокировка, выход |
| Статус ответа ИИ | до конца озвучки | окончание речи / новая команда |
| Озвучка ответа | до конца текста | **«Иван хватит»** прерывает |
| История диалога с ИИ | 1 последняя реплика (экономия памяти) | — |

## 3. Помощник ИИ (главное)

| Скажите | Что делает |
|---|---|
| «Иван, думай …» / «спроси …» | Вопрос к ИИ. Можно **одной фразой**: «Иван думай сколько будет семью восемь» → сразу ответ. Без вопроса — подождёт следующую фразу |
| «Иван, поразмышляй …» / «порассуждай …» | Развёрнутое рассуждение |
| «Иван, поговорим» | Режим диалога: говорите свободно, Иван помнит контекст. Выход — «хватит» |
| «Иван, подробнее» | Развернуть последний короткий ответ |
| «Иван, сформулируй …» | Составит текст и **вставит в поле ввода**, вслух «Сгенерировал текст» |
| «Иван, хватит» (во время речи) | **Прервать озвучку** ответа |

**Что ИИ считает/знает сам (мгновенно, без модели):**
- арифметика — «восемь плюс два минус один» = 9 (точно, в коде);
- время, дата, день недели, заряд батареи — из устройства;
- погода — «Иван думай какая погода» (Open-Meteo, нужен интернет + геолокация).

**Длина ответа** подбирается под вопрос: простое — коротко, «порассуждай/объясни» — развёрнуто. Мало — «подробнее».

**Что отвечает, а что нет:** свободно рассуждает, советует, придумывает, делится мнением. «Не уверен» — только про факты, которые не может знать (свежие новости, точные цены, номера законов, статистику, даты); их не выдумывает.

## 4. Звонки

| Скажите | Что делает |
|---|---|
| «Иван, ответь» / «прими» | Принять вызов |
| «Иван, отбой» / «сбрось» / «заверши» / «трубку» | Завершить вызов |

**Логика звонка:**
- Во время вызова (мобильного и в приложении) — **все команды только через «Иван…»**, чтобы речь в разговоре ничего не запускала.
- Сброс пробует по очереди: система → кнопка на экране (по тексту/описанию и id) → медиа-клавиша → тап по красной кнопке.
- **Шумоподавление** на время мобильного звонка **автоматически выключается** (его микрофонный режим конфликтует с вызовом) и возвращается после. Поэтому держать NS включённым теперь можно.

## 5. Полная таблица команд

### Ввод текста
напиши/диктовка (режим), цифры/номер телефона, выдели, **очисти** (Backspace через клавиатуру — работает и в Заметках), удали/сотри, вставь, копируй, ввод/отправь/энтер.

### Навигация и жесты
назад, домой, недавние, шторка, быстрые/панель, заблокируй, скриншот, листай/крути (вверх/вниз), вверх/вниз/влево/вправо (+«чуть»), до конца/в начало, смахни N, перетащи N M, тап/центр, нажми N, нажми «текст», номера, сетка.

### Приложения и связь
открой/запусти «имя», позвони/набери «имя», голосовое N/запиши, отправь.

### Звук и питание
громкость 1–10, громче/тише, без звука, беззвучно, медиа, пауза/играй/следующий/предыдущий, скрой/убери оверлей, помощь.

### Спец
привет (разбудить при выкл. экране), SOS/спасите/тревога, молчи/замри (пауза), слушай/продолжи.

## 6. Особые случаи и зависимости

- **Экран выключен:** только «Иван» + (громче/тише/без звука/пауза/играй/следующий) и «Иван ответь/отбой». Остальное игнор.
- **Озвучка:** во время речи слышит только «Иван + хватит/стоп» (прерывание); прочее игнор (анти-петля). Лучше работает с включённым шумоподавлением (эхоподавление).
- **Размышление ИИ:** пока думает — всё игнор.
- **Команды распознаются по словарю-грамматике** — новые командные слова надо добавлять в `Vocabulary.kt`, иначе Vosk их «не услышит». Захват вопроса к ИИ и диктовка — свободные, без словаря.
- **Общие слова в разных контекстах:** «стоп» — выход из диктовки / отмена ИИ / пауза медиа / прерывание озвучки; «отмена» — выход из диктовки / скрыть оверлей.
- **Порядок разбора:** озвучка(прерывание) → размышление ИИ → захват ИИ → звонок(только «Иван») → экран выключен → медиа → диктовка → команда.
- **Память устройства:** модель ИИ (`.task`) при первом запросе распаковывается в кэш (≈ размер модели, разово, чистится безопасно).
