# ГолосРуки — карта команд, состояний и логики (v6.39)

Документ описывает, **как устроена логика**: какие есть режимы, когда команда срабатывает,
нужно ли слово активации «Иван», что команда делает/симулирует, сколько держится и как выключается.

---

## 1. Состояния и режимы

Приложение всегда в одном из двух **базовых состояний**, поверх которых включаются **под-режимы** (флаги).

| Состояние / режим | Что значит | Как входим | Как выходим | Слышит без «Иван»? |
|---|---|---|---|---|
| 😴 **Сон** (ASLEEP) | Ждёт только слово активации | Бездействие ~30 c (настраивается) | Слово «Иван» → Бодрствование | Только «Иван» (и при выкл. экране — пара быстрых, см. §4) |
| 🎙 **Бодрствование** (AWAKE) | Принимает команды без «Иван» | Сказать «Иван» | Бездействие → Сон | Да, все команды |
| ✍️ **Диктовка** | Речь → текст в поле | Команда «напиши/диктовка» | «готово», «конец», «стоп», «отмена», «Иван» | Да (всё — текст) |
| 🔢 **Диктовка цифрами** | Речь → цифры | Команда «цифры/номер телефона» | Так же, как диктовка | Да |
| 🎵 **Медиа-режим** | Пауза/играй/треки/листание без «Иван» | Команда «медиа» (переключатель) | Команда «медиа», блокировка, выход из приложения | Да (только медиа/листание) |
| 📞 **Звонок** | Идёт вызов | Система телефонии / VoIP | Завершение звонка | **Нет — только через «Иван»** |
| 🧠 **Размышление ИИ** | Модель генерирует ответ | После вопроса к ИИ | Приход ответа | Нет (всё игнор) |
| ❓ **Захват ИИ-запроса** | Ждёт текст вопроса | Команда «думай/спроси/сформулируй» | Фраза-вопрос, «стоп», «отмена», «Иван», или сон | Да (следующая фраза = вопрос) |
| ⏸️ **Пауза** | Распознавание приостановлено | Команда «молчи/замри» | Команда «слушай/продолжи» | Только «слушай/продолжи»/«Иван» |

## 2. Таймауты и удержания

| Что | Сколько держится | Чем сбрасывается |
|---|---|---|
| Бодрствование → Сон | **30 c** бездействия (настраивается в «Настройки → Сон») | Любая команда продлевает |
| Медиа-режим | **Бессрочно**, до явного выключения | «медиа», блокировка, выход из приложения |
| Статус ответа ИИ на экране | **До конца озвучки** ответа (или по времени чтения, если озвучка выкл.) | Окончание TTS / новая команда |
| Пауза распознавания при озвучке (TTS) | ~1.8 c + по длине текста | Автоматически после проговаривания |
| Уведомление сопряжения adb | 12 c | Автоисчезает |

## 3. Полная таблица команд

«Иван» нужен, если вы в **Сне**. В **Бодрствовании** — без «Иван». В диктовке всё, кроме слов выхода, становится текстом.

### Помощник ИИ
| Скажите | Команда | Что делает | Удержание |

## 4. Особые случаи

- **Экран выключен:** работают только `Иван` + (громче/тише/без звука/пауза/играй/следующий/предыдущий) и управление звонком `Иван + ответь/отбой`. Остальное игнорируется (экономия и защита от ложных срабатываний).
- **Медиа-режим + сон:** при простое включается сон, но медиа-режим сохраняется и вернётся после пробуждения.
- **Размышление ИИ:** пока модель думает, всё распознавание игнорируется, чтобы шум не сбивал.
- **Захват ИИ-запроса важнее медиа/экрана:** после «думай/спроси» следующая фраза гарантированно уходит в вопрос.

## 5. Зависимости (что на что влияет)

Эти связи важны при добавлении/изменении команд:

- **Слово выхода из диктовки** (`готово/конец/стоп/отмена/Иван`) — если добавить команду с этими словами, она может прерывать диктовку.
- **`стоп`** используется и для выхода из диктовки, и как `MediaPause`/отмена захвата ИИ — одно слово, три контекста.
- **`отмена`** — выход из диктовки и `HideOverlay`.
- **Медиа-режим** перехватывает медиа/листание/назад/домой **без «Иван»** — пока он включён, эти слова не дойдут до обычного разбора.
- **`inCall()`** (идёт звонок) — переводит разбор в режим «только через Иван»; если определение звонка ложно сработает, обычные команды потребуют «Иван».
- **Порядок разбора в onResult:** озвучка → размышление ИИ → захват ИИ → звонок → экран выключен → медиа → диктовка → пробуждение/сон → команда. Верхние перехватывают раньше нижних.
- **`сформулируй`/диктовка/очистка** используют невидимую клавиатуру (IME). Если авто-переключение клавиатуры выключено (нет разрешения), они откатываются на спец-возможности и в холсте (Samsung Notes) могут не сработать.
